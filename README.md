# 🏀 NBA Dashboard, 2025-26 (Power BI)

## 📌 Overview

This project is an interactive Power BI dashboard that analyzes **NBA game performance during the regular season 2025-26**, focusing largely on player performace during the season along with team results, scoring trends, and live scores.

The dashboard provides statistical insights into how players are performing, helping identify patterns in scoring, outcomes, and overall performance.

---

## 🎯 Objective

* Analyze **player performance** across NBA
* Compare **wins vs losses** using key metrics
* Understand scoring trends across matches
* Provide a visual summary of team performance

---

## 📂 Data Source

Data is retrieved using the Python library:

* nba_api

### Key Fields Used:

* Team ID
* Player ID
* Game Date
* Matchup
* Win/Loss (WL)
* Points Scored (PTS)

---

## ⚙️ Data Preparation

* Extracted game data using Python scripts
* Processed and cleaned data for analysis
* Derived fields:

  * Final Score (combined from both teams)
  * Win/Loss indicators
  * Team-level metrics

---

## 🛠️ Tools & Technologies

* Power BI
* Python (data extraction using NBA API and selenium package)
* DAX (for calculated measures and KPIs)

---

## 📊 Key Features & Visuals

### 🏆 Win vs Loss Analysis

* Visual comparison of team performance
* Highlights win rate and patterns

### 📈 Player Scoring Analysis

* Tracks points scored across games
* Identifies player's recent form 
* Player's shot making and shot efficiency

### 📊 Match-Level Insights

* Displays matchup details and results
* Includes game outcomes and scores
* Visualize scoring patterns of each game

---

## 🧠 Key Insights

* Teams show varying performance across different matchups
* Certain teams consistently achieve higher scores
* Win/loss patterns highlight performance consistency
* Game-level analysis helps identify trends and anomalies
